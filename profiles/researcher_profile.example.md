# Researcher Profile

## Research Goal

Build a hot-first but relevance-aware research digest that turns daily signals into a few strong ideas with initial plans.

## Primary Research Directions

### 1. RL And Policy Learning
- Reinforcement learning
- On-policy / off-policy learning
- Post-training
- Policy distillation
- RLHF / RLVR

### 2. Generative Modeling
- Diffusion models
- Autoregressive generation
- Flow / latent generative modeling

### 3. MLLM And Unified Gen-Understand Models
- MLLM
- Generation-understanding unification
- Multimodal reasoning

### 4. Agents
- Tool-using agents
- Planning and execution
- Autonomous workflows
- Multi-agent systems

### 5. Video And Embodied
- VideoLLM
- Video generation / understanding
- Embodied multimodal agents

### 6. Hot But Worth Tracking
- Attention alternatives
- Systems breakthroughs
- Architecture shifts
- Viral papers with transfer potential

## Digest Preferences

- Prefer hot-first candidate pools with relevance-aware reranking.
- Keep both strong-related items and globally hot weak-related items.
- End with 1-3 evidence-backed ideas, each with an initial feasible plan.

## Summary Style

For each important item, include:

1. TLDR
2. Problem
3. Method core
4. Pipeline / training recipe / system workflow
5. What is new
6. Why it matters
7. How it relates to the research directions

## Idea Requirements

Each idea should include:

- basis / evidence
- core insight
- initial plan
- minimum experiment
- novelty
- feasibility
